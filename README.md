## О проекте

**CryptixVPN** — это VPN-сервис нового поколения на базе протоколов **V2Ray**, **VLESS** и **Xray**, управляемый через **Telegram Bot API**. Сервис предоставляет зашифрованный туннель для обхода интернет-цензуры и блокировок в России, Беларуси и Казахстане.

Вся работа с подпиской ведётся прямо в Telegram: покупка, продление, смена сервера, получение конфигурационного ключа — без регистрации на сторонних сайтах.

**Технологии:** Xray-core, VLESS + XTLS-Reality, V2Ray, Telegram Bot API, автоматический биллинг через СБП и USDT TRC-20.
